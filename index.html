Jasmine Test

Ein Beispiel

function reverse(val) {
  if (typeof val === 'string') {
    return val.split('').reverse().join('');
  }
  return val;
}

Jasmine Test

Ein Beispiel

describe('reverse', function() {
  it('should reverse strings', function() {
    expect(reverse('hallo')).toEqual('ollah');
    expect(reverse('bye')).toEqual('eyb');
  });
})
  • expect
    Beschreibt einen Teil/Funktionalität der Software
  • it
    Beschreibt das erwartete Verhalten
  • toEqual
    Matcher - Der erwartete Rückgabewert

Jasmine Test

Ein Beispiel

describe('reverse', function() {
  it('should reverse strings', function() {
    expect(reverse('hallo')).toEqual('ollah');
    expect(reverse('bye')).toEqual('eyb');
  });

  it('should ignore value types other than strings', function() {
    expect(reverse(true)).toBe(true); // reverse(true) === true
    expect(reverse(undefined)).toBe(undefined);
  });
})

Test Runner

Karma Test Runner

  • Node.JS basiert - Open Source auf Github
  • 2011 gestartet
  • Projekt von Google/AngularJS Mitarbeiter
  • Tests können in allen beliebigen Browsern parallel ausgeführt werden:
    • Destop
    • Smartphones
    • Tablets
    • ...
  • Testergebnis gebündelt in der Konsole sichtbar
  • Kompatibel mit den gängigen Test Frameworks
    (Jasmine, QUnit, Mocha)
  • Ausführung auf CI Server möglich (Stichwort: Continous Integration)
